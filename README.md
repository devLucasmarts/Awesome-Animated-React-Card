## Run the project
<br />
Git clone
<pre>
    git@github.com:martsDev/Awesome-Animated-React-Card.git
</pre>

<br />

Access project directory
<pre>
    cd Awesome-Animated-React-Card
</pre>

<br />

Run npm install
<pre>
    npm install
</pre>

<br />

Run the project
<pre>
    npm start
</pre>
